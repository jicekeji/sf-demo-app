# sf-demo-app

A tiny static web app **built from 0→1 by [SuperFactory](https://serion.cc/sf/)** as an end-to-end test of the
factory's full lifecycle (ideation → spec → plan → build → quality_gate → deploy → operate).

- Hosted on **Render** (static site, auto-deploys from `main`).
- Monitored by **UptimeRobot** (uptime) — alerts flow back into the factory's incident inbox.
- The skeleton here is intentionally minimal; the real features are added by factory-generated PRs.
